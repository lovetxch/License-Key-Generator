<h1 align="center">License Key Generator</h1>

###

<p align="center">My first Project: A simple license key generator made in Python with all essential features, perfect for generating license keys for your projects. All keys are highly randomized. The code is easy to understand and well commented. Adding new features is simple, just set the boolean variable more_features to True in main.py.<br><br>Quick showcase video: https://youtu.be/2S-yFoHzohM</p>

<h2 align="left">Features:</h2>

###

<p align="left">• Save keys to a .txt file<br><br>• Add prefix or suffix (text at the beginning or end of each key)<br><br>• Set key length<br><br>• Set the number of keys to generate<br><br>• Numbered keys<br><br>• Character type options: Letters & Numbers, Letters only, Numbers only<br><br>• Different formatting options: Random, Uppercase, Lowercase<br><br>• Remove Dash<br><br>• Option to add your own custom features</p>

###

<h2 align="left">Setup</h2>

###

<p align="left">1. Install the latest version of Python from: <code>https://www.python.org/downloads</code>/<br><br>2. Open the Command Prompt and run: <code>python -m pip install --upgrade pip</code> to update pip<br><br>3.  Install PyInstaller by running: <code>python -m pip install pyinstaller</code><br><br>4. Download the Key Generator source code from GitHub.<br><br>5. Open CMD<br><br>6. Navigate to the project folder by typing: <code>cd path to the project folder <path to the project folder></code><br><br>7. Once you are in the correct folder, build the executable with the command:<br><code>pyinstaller --onefile --windowed --icon=exe_icon.ico --add-data "exe_icon.ico;." main.py</code><br><br>8. If the Command Prompt indicates that the build was successful, you can find the .exe file in the dist folder inside your project directory.</p>
